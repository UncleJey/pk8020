# pk8020
реинжениринг корвет ПК 8020

* процессор КР580ВМ80А на тактовой частоте 2,5 МГц быстродействие - 625 тысяч операций в секунду.
* ПЗУ 96КБ (8КБит х 3 ROM + 64КБит х 8 RAM)
* ГЗУ 192КБайт ()
* клавиатура 80 клавиш
* звукогенератор
* НГМД
* параллельный порт
* последовательный порт
* магнитофон
* внешние модули
* 2 джойстика
